# Deployments

This file is the source of truth for published `aeqi-solana` program IDs.

## Status

No production deployment has been recorded from this repository.

The IDs in `Anchor.toml` are localnet development IDs unless a release entry
below says otherwise.

## Release Template

Copy this section for every deployed release.

```text
release:
commit:
cluster:
deployer:
upgrade authority:
immutable:
anchor version:
solana cli version:
verifiable build image:

programs:
- name:
  program id:
  artifact:
  local executable hash:
  deployed executable hash:
  idl hash:
```
