## Do / don't

- **Do** use Inter for everything chrome-related. If it could run in a
  settings row, it's Inter.
- **Do** keep Exo 2 for moments that deserve monumentality. Overuse
  dilutes it.
- **Don't** set a new font family anywhere. Four is the budget.
- **Don't** hand-tune letter-spacing on body text. The defaults are
  calibrated.
