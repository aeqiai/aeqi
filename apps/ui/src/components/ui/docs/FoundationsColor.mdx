# Color

Click any swatch to copy its token. Values are read live from `:root`, so
this page tracks `packages/tokens/src/tokens.css` — never hardcode a hex.

## Surfaces

Four-tier elevation ladder (settled 2026-04-27, v5.1). Paper outside,
card-tier inside, elevated above the card for popovers and pills, ink
sheet for inversion. The inset family lives inside the card.

Depth comes from value contrast across the ladder, not from drop shadows.

## Accent

The accent is near-black graphite. It owns the wordmark, primary CTAs,
links, focus rings, and active state. At ~2% surface area, it feels
authoritative. At more, it feels heavy.

## Status

Status colors are the only chromatic exception outside the accent. They
appear minimally — a dot, a tint, subtle text color — never a full-bleed
wash.

## Do / don't

- **Do** carry hierarchy with opacity on ink, not with new greys.
- **Do** reserve jade for success. Don't use it for decoration or brand.
- **Do** keep the accent at ~2% of pixel area. More reads as chrome.
- **Don't** hardcode hex values in component CSS. Add a token to
  `@aeqi/tokens` first, then alias it in `primitives.css`.
- **Don't** use a status color as a full-bleed background. A dot or a 10%
  tint is almost always enough.
