## Design Context

### Users
Founders and technical operators building AI-first companies. They deploy autonomous agents that handle engineering, research, operations — continuously and in parallel. They're sophisticated, impatient, and expect tools that match their ambition. The interface is a control plane for an autonomous workforce, not a chatroom.

### Brand Personality
**Precise. Autonomous. Quiet.**

The æ sigil is the brand mark. The product is infrastructure — it should feel like infrastructure. No personality theater, no emoji-laden copy. The interface communicates through structure and typography, not decoration. When something is working, it's silent. When something needs attention, it's clear.

### Aesthetic Direction
**Monochromatic brutalist minimalism with typographic precision.**

- **Palette**: Pure black-on-white zinc. No accent colors. Status colors (green/red/amber) are functional only. Everything else is shades of black at varying opacity.
- **Typography**: Inter for prose, JetBrains Mono for data/code/labels. Type hierarchy does all the work.
- **Space**: Generous whitespace. Content breathes. Density comes from information, not cramming.
- **Surfaces**: Flat. No gradients, no glassmorphism, no glows. Borders at 8% opacity. Shadows only for elevation.
- **Motion**: Near-invisible. Fade-in at 200ms. No bouncing, no spring physics. The æ pulse is the one allowed animation.
- **Theme**: Light mode only.

**References**: Linear (information density), Vercel (monochrome discipline), Bloomberg Terminal (data-first), Stripe (institutional quality)

**Anti-references**: Slack (colorful noise), Discord (dark gaming), typical SaaS dashboards (gradient cards, metric tiles), cyan-on-dark "AI" aesthetic, Notion (playful/emoji), crypto dashboards (neon/gradients)

### Design Principles
1. **Information over decoration.** Every pixel communicates. If removing an element loses no information, remove it.
2. **Typography is the interface.** Size, weight, opacity, and font family create all hierarchy. A 10px mono label at 30% opacity IS the UI.
3. **Silence means working.** Default state is calm. Activity indicators appear during operations and disappear when done.
4. **Density on demand.** Start sparse, reveal detail through interaction. The first view is the simplest possible.
5. **Monochrome is not monotone.** Black at 85%, 50%, 30%, 15%, 8%, 4% — six levels of emphasis, one color.
6. **Respect the user.** No onboarding slides, no tooltips for obvious things, no patronizing empty states.
